# 🎮 ModGG Client

**Легальный клиентский мод для Minecraft 1.21.4 (Fabric)**

Разработчик: **kats676**

Крутые визуальные функции без читерства. Работает на любых серверах!

---

## ⚡ Функции

| Клавиша | Функция | Описание |
|---------|---------|----------|
| `RSHIFT` | Меню | Открыть меню настроек |
| `H` | CreateHue | Редактор рук (размер и позиция) |
| `L` | LavaCam | Чёткое зрение под лавой |
| `F6` | FreeCam | Свободная камера |
| `K` | Effects | Фейерверк при убийстве |
| `P` | Glow Players | Подсветка видимых игроков |

---

## 📦 Установка

1. Установи [Fabric Loader](https://fabricmc.net/)
2. Скачай [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
3. Скачай `ModGG-Client.jar` из раздела Releases
4. Положи оба файла в папку `.minecraft/mods/`
5. Запускай Minecraft с профилем Fabric!
